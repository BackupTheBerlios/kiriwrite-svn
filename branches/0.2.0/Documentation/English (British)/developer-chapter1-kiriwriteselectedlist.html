1.29 kiriwrite_selectedlist

kiriwrite_selectedlist gets the list of selected pages and processes them into an array which is used by the kiriwrite_page_multidelete, kiriwrite_page_multimove, kiriwrite_page_multicopy, kiriwrite_page_multiedit and kiriwrite_compile_makepages.

Parameters:

kiriwrite_selectedlist();

Usage:

@filelist = kiriwrite_selectedlist();
