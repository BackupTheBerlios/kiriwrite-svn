Introduction

This documentation provides a tutorial which allows the main functions of Kiriwrite to be learned.

The tutorial consists of the following:
  • Create a 3 page website using the same layout on all of the pages.
  • Create a 3 page website using the same layout on all of the pages but with certain words changed using the filters database.
  • Create a 10 page website with 5 pages using one layout and another 5 pages using another layout.
