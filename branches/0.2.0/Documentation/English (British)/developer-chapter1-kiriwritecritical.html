1.35 kiriwrite_critical

kiriwrite_critical processes critical errors when using Kiriwrite.

Parameters:

kiriwrite_critical(errortype);

Usage:

kiriwrite_critical(“generic”);

errortype
Specifies the critical error message to use.

For more information on the error messages used look at the kiriwrite_critical subroutine in the Kiriwrite source code.
