Welcome
This is the documentation for Kiriwrite and it is split into three parts:
  • User Documentation
    The User Documentation provides information on how to install and use your Kiriwrite installation.
  • Tutorial Documentation
    The Tutorial Documentation provides some tutorials on how to use certain featurs of Kiriwrite.
  • Developer Documentation
    The Developer Documentation provides information on how to create additional presentation modules, database modules and information on translating Kiriwrite into another language.
The Kiriwrite Documentation Guide provides information about the symbols that appear while reading the documentation.
