1.13 kiriwrite_database_delete

kiriwrite_database_delete deletes a database. If the confirm value is not given then it will write a form asking if the database should be deleted. If a database filename is not specified then an error is returned.

kiriwrite_database_delete will return data that is intended for kiriwrite_output_page.

Parameters:

kiriwrite_database_delete(database, confirm);

Usage:

$pagedata = kiriwrite_database_delete(“database”);
$pagedata = kiriwrite_database_delete(“database”, 1);

database
Specifies the database to delete. This is required each time this subroutine is called.

confirm
Specifies if the action to delete a database should be confirmed by passing a '1'.
