3.6 geterror

Gets the error message stored in the $error scalar (or extended error message stored in the $errorext scalar).

Parameters:

$kiriwrite_dbmodule->geterror(extendederror);

Usage:

$errormessage = $kiriwrite_dbmodule->geterror();
$extendederrormessage = $kiriwrite_dbmodule->geterror(1);

extendederror
Specifies if the extended error information should be returned.
