1.32 kiriwrite_processfilename

kiriwrite_processfilename processes a name and turns it into a filename that can be used by Kiriwrite.

Parameters:

kiriwrite_processfilename(text);

Usage:

$filename = kiriwrite_processfilename(“For the 3rd time already”);

text
Specifies the text to create a filename with.
