1.24 kiriwrite_compile_all

kiriwrite_compile_all compiles all of the available databases. It writes the list of databases available to compile which will be passed to the kiriwrite_compile_makepages subroutine.

kiriwrite_compile_all will return data that is intended for kiriwrite_output_page.

Parameters:

kiriwrite_compile_all();

Usage:

$pagedata = kiriwrite_compile_all();
