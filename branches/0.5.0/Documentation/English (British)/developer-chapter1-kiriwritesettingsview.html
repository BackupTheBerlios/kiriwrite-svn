1.27 kiriwrite_settings_view

kiriwrite_settings_view lists the settings that are currently being used in Kiriwrite.

kiriwrite_settings_view will return data that is intended for kiriwrite_output_page.

Parameters:

kiriwrite_settings_view();

Usage:

$pagedata = kiriwrite_settings_view();
