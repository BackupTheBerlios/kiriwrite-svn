Chapter 2: Operation

Now that Kiriwrite is installed and working, this chapter is about operating Kiriwrite. This chapter is split into five parts:
  • Databases, how to add a database, edit a database and delete a database.
  • Pages, Selecting a database, creating, editing and deleting pages, deleting, moving, copying and editing multiple pages.
  • Filters, adding a filter, editing a filter and deleting a filter.
  • Templates, adding a template, editing a template and deleting a template.
  • Compiling, compiling one, selected or all databases and cleaning the output directory.
